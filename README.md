<h1>Odysseus</h1>

## How to run

```
npm install
npm start
```
Open [http://localhost:2567](http://localhost:2567) in your browser.

### if running from WSL2

> netsh interface portproxy add v4tov4 listenport=2567 listenaddress=0.0.0.0 connectport=2567 connectaddress=172.30.80.72
